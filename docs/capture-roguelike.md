# 고양이 포획 로그라이크 상세 기획

> 최종 수정일: 2026-04-04

## 레퍼런스

- **20 Minutes Till Dawn** — 핵심 게임플레이 참고
  - 고정 맵에서 몰려오는 적을 상대하는 서바이벌 슈터
  - 레벨업 시 스킬 선택으로 빌드 구성
  - 제한 시간 동안 생존하면 클리어

## 기본 구조

| 항목 | 내용 |
|------|------|
| 장르 | 서바이벌 슈터 (20 Minutes Till Dawn 스타일) |
| 시점 | 탑다운 2D |
| 맵 | 고정 크기, 경계 있음 |
| 조작 | 이동 + 회피 + 공격 |
| 클리어 조건 | 제한 시간 동안 생존 + 보스 고양이 격파 |
| 실패 조건 | HP가 0이 되면 포획 실패 |

## 플레이 타임 단계

플레이어가 여정 출발 시 난이도를 선택한다. 단계가 높을수록 희귀 고양이 출몰 비율이 증가한다.

| 단계 | 시간 | 난이도 | 희귀 고양이 출몰 비율 |
|------|------|--------|-------------------|
| 짧은 여정 | 5분 | 쉬움 | 낮음 — 일반 고양이 위주 |
| 보통 여정 | 10분 | 보통 | 중간 — 희귀 고양이 등장 |
| 긴 여정 | 20분 | 어려움 | 높음 — 고희귀 고양이 등장 |

> 구체적인 출몰 비율은 밸런싱 단계에서 조정

## 전투 흐름

```
여정 시작 (시간 단계 선택)
        │
        ▼
  일반 고양이 웨이브
  (시간 경과에 따라 강해짐)
        │
        ▼
  레벨업 → 스킬 선택
        │
        ▼
  ... 반복 ...
        │
        ▼
  보스 고양이 등장 (제한 시간 종료 직전)
        │
   격파 │    │ 사망
        ▼    ▼
   포획 성공  포획 실패
```

## 무기 목록

고양이를 "때린다"는 느낌보다는 **장난스럽게 상대한다**는 톤을 유지한다.
맞아도 아프지 않을 것 같은, 죄책감 없는 무기들로 구성.

| 무기 | 설명 | 공격 타입 |
|------|------|----------|
| 새총 | 작은 돌멩이를 쏘는 클래식 새총 | 원거리 단발 |
| 탱탱볼 | 통통 튀는 고무공을 던짐 | 원거리, 벽 반사 |
| 나무가지 | 가벼운 나뭇가지로 휘두름 | 근거리 휘두르기 |
| 실뭉치 | 실타래를 풀어서 던짐 | 원거리 관통 |
| 레이저 포인터 | 레이저로 고양이를 현혹 | 원거리 지속 데미지 |
| 물총 | 물줄기를 발사 | 원거리 연사 |
| 깃털 장난감 | 깃털 낚싯대를 휘두름 | 근거리 넓은 범위 |
| 종이공 | 구겨진 종이를 던짐 | 원거리, 약한 유도 |
| 비눗방울 | 비눗방울을 날려 터뜨림 | 원거리 범위 공격 |
| 마따따비 폭탄 | 마따따비 가루를 뿌려 범위 피해 | 설치형 범위 공격 |

> 무기 종류는 추후 확장 가능. 밸런스 수치는 개발 단계에서 조정.

## 성장 시스템

20 Minutes Till Dawn과 유사하게 **런 내 성장** 방식을 채택한다.

### 런 내 성장 (로그라이크)
- 적 처치 시 경험치 획득 → 레벨업
- 레벨업 시 **3개 중 1개 스킬 선택** (랜덤 제시)
- 스킬 예시 (후보):
  - 공격 속도 증가
  - 투사체 개수 추가
  - 이동 속도 증가
  - 최대 HP 증가
  - 투사체 크기 증가
  - 넉백 강화
  - 무기 고유 강화 (예: 탱탱볼 반사 횟수 증가)

> 런 간 영구 성장 시스템은 추후 검토

## 적 (고양이) 패턴

고양이들이 플레이어를 향해 다가오며, 종류에 따라 행동이 다르다.

| 행동 유형 | 설명 | 예시 |
|----------|------|------|
| 돌진형 | 플레이어를 향해 직선 돌진 | 치즈 고양이 |
| 배회형 | 주변을 어슬렁거리며 가끔 돌진 | 회색 고양이 |
| 원거리형 | 멀리서 털뭉치/헤어볼 발사 | 앙고라 (흰색 고양이) |
| 보스 | 고유 패턴 보유 | 각 고양이별 고유 보스전 |

> 각 고양이별 상세 패턴은 개발 단계에서 구체화
