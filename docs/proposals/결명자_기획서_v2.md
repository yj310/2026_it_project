## 1. 서비스 개요

### 가. 서비스 제안 배경

공모전의 목표는 실제 출시와 운영이 가능한 IT 서비스를 만드는 것이다. 따라서 캣치냥은 처음부터 큰 규모의 게임을 만들기보다, 웹에서 바로 접속할 수 있고 성과지표를 측정할 수 있는 최소 기능 게임으로 출시하는 것을 1차 목표로 한다.

고양이 캐릭터는 대중적으로 호감도가 높고, 수집 욕구를 만들기 쉽다. 로그라이크 전투는 짧은 시간 안에 반복 플레이를 유도할 수 있어 웹 게임 MVP에 적합하다. 캣치냥은 “전투에 성공하면 고양이를 얻는다”는 단순한 핵심 규칙을 중심으로, 사용자가 바로 이해하고 반복해서 플레이할 수 있는 고양이 포획 게임을 제공한다.

초기 버전에서는 미니게임, 꾸미기, 인앱결제 등 확장 기능을 제외하고, 서비스 검증에 꼭 필요한 핵심 기능만 구현한다. 이후 사용자 지표와 피드백을 바탕으로 재미 요소를 단계적으로 추가한다.

### 나. 서비스 핵심 내용

캣치냥은 웹에서 실행되는 고양이 포획 로그라이크 게임이다. 사용자는 제한 시간 동안 적의 공격을 피하고, 경험치를 얻어 능력을 강화한 뒤, 마지막에 등장하는 보스 고양이를 공략한다. 성공하면 해당 고양이를 도감에 등록할 수 있다.

핵심 루프:

```text
게임 접속
  -> 포획하기 선택
  -> 로그라이크 전투 시작
  -> 생존 및 성장
  -> 보스 고양이 공략
  -> 고양이 획득
  -> 도감 확인
  -> 다시 도전
```

MVP 범위:

- 웹 URL로 접속 가능한 게임
- 포획 전투 1개 모드
- 플레이어 이동, 자동 공격, 체력, 경험치, 레벨업
- 레벨업 시 능력 선택 3종
- 보스 고양이 1종
- 포획 가능한 고양이 3종
- 포획 성공 결과 화면
- 고양이 도감
- Google Analytics 기반 성과지표 측정

MVP에서 제외하는 기능:

- 미니게임
- 홈 화면 꾸미기
- 고양이 의상 및 액세서리
- 슬롯 확장 및 유료 재화
- Android, Steam 정식 출시
- 계정 시스템 및 서버 저장

| 고양이 콘텐츠 선호 사용자 | 귀여운 캐릭터, 수집 요소에 반응 | 고양이 획득과 도감 기능이 핵심 동기가 된다. |

## 2. 시장 조사 및 분석

### 가. 시장 규모

게임 시장은 웹, 모바일, PC 등 다양한 플랫폼에서 성과지표 측정이 가능하다. Newzoo의 2025 Global Games Market Report에 따르면 2025년 글로벌 게임 매출은 1,888억 달러 규모로 전망되며, 모바일 게임 매출은 1,030억 달러로 전체의 55%를 차지한다. 모바일 플레이어 수는 약 30억 명으로 추정된다.

캣치냥은 장기적으로 모바일과 PC 확장이 가능하지만, 공모전 기간에는 웹 MVP를 우선 출시한다. 웹은 설치 과정이 없어 진입 장벽이 낮고, 학과 커뮤니티와 SNS를 통한 링크 공유가 쉽다. 또한 Google Analytics를 통해 DAU, 신규 사용자, 재방문율, 플레이 이벤트 등 성과지표를 빠르게 측정할 수 있다.

### 다. 경쟁사 대비 차별화 전략

캣치냥은 고양이 수집과 짧은 로그라이크 전투를 결합한다. 사용자는 단순히 보상을 기다리는 것이 아니라, 직접 조작하고 생존에 성공해야 고양이를 얻는다. 이 구조는 MVP 단계에서도 “포획에 성공하고 도감이 채워지는 재미”를 만들 수 있다.

차별화 전략:

- 설치 없이 웹에서 바로 실행
- 한 판 3~5분 내외의 짧은 플레이
- 고양이 획득이라는 명확한 보상
- 도감을 통한 수집 목표 제공
- 향후 미니게임, 꾸미기, 모바일 출시로 확장 가능한 구조

#### 주요 시나리오

1. 사용자는 웹 링크에 접속한다.
2. 시작 화면에서 `포획하기` 버튼을 누른다.
3. 전투 화면에서 캐릭터를 이동해 적의 공격을 피한다.
4. 적을 처치해 경험치를 얻고 레벨업한다.
5. 레벨업 시 능력 1개를 선택한다.

9. 도감에서 보유 고양이를 확인하고 다시 도전한다.

## 4. 수익모델

### 가. 수익화 방안

MVP 단계에서는 수익화보다 실제 출시, 사용자 확보, 지표 측정을 우선한다. 초기 버전은 무료 웹 게임으로 공개하고, 사용자가 핵심 루프를 반복해서 플레이하는지 검증한다.

추후 수익화는 게임 진행을 방해하지 않는 선택형 모델로 확장한다.

- 보상형 광고: 추가 도전 기회, 보상 2배
- 꾸미기 아이템: 고양이 의상, 배경, 장식
- 슬롯 확장: 보유 가능한 고양이 수 증가
- 후원형 결제: 웹 버전에서 개발 응원 결제

### 나. 단기 수익 모델, 대회 기간 기준

| 수익 모델 | 적용 여부 | 설명 |

| 무료 공개 | 적용 | 공모전 기간에는 사용자 유입과 성과지표 확보를 우선한다. |
| 후원형 결제 | 선택 적용 | 구현 여유가 있을 경우 개발 응원 버튼을 추가한다. |
| 광고 | 보류 | MVP 안정화 후 보상형 광고 적용을 검토한다. |
