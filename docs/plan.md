## 2. 핵심 컨셉

플레이어는 다양한 고양이를 **로그라이크 전투로 포획**하고, 포획한 고양이와 함께 **2010년대 인기 게임을 패러디한 미니게임**을 플레이한다.

### 컨셉 키워드
- 수집 욕구: 다양한 종류의 귀여운 고양이 수집
- 도전감: 하드코어 로그라이크 포획전
- 향수: 2010년대 유행 게임 패러디 미니게임

## 3. 게임 루프

```
         │
         ▼

│   미니게임 플레이  │  ← 2010년대 패러디 게임
└─────────────────┘
```

## 4. 핵심 시스템

### 4.1 고양이 포획 (로그라이크)

- **시점**: 플레이어(사람) 캐릭터가 화면 중앙에 위치
- **공격 패턴**: 사방에서 나뭇잎, 나무가지 등 자연물 기반 공격이 날아옴
- **스테이지 구성**: 일반 스테이지 → 보스 고양이 등장
- **승리 조건**: 보스 고양이를 격파하면 포획 성공
- **패배 조건**: 도중 사망 시 포획 실패
- **플레이 타임**: 세션당 10분 이상 (하드코어 난이도)
- **성장 요소**: 스킬을 쌓아가며 점점 강한 고양이에 도전

### 4.2 고양이 슬롯 시스템

- 보유 가능한 고양이 수에 **슬롯 제한**이 있음
- 슬롯 확장 방법:
  - **유료 재화** (인앱결제)
  - **희귀 재화** (게임 내 획득)
- 슬롯 제한은 수집 전략과 과금 유도의 핵심 장치

### 4.3 미니게임

2010년대 유행했던 게임들을 고양이 테마로 패러디한 미니게임 컬렉션.
