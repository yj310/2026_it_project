## 2. 핵심 컨셉

플레이어는 다양한 고양이를 **로그라이크 전투로 포획**하고, 포획한 고양이와 함께 **2010년대 인기 게임을 패러디한 미니게임**을 플레이한다.

### 컨셉 키워드
- 수집 욕구: 다양한 종류의 귀여운 고양이 수집
- 도전감: 하드코어 로그라이크 포획전
- 향수: 2010년대 유행 게임 패러디 미니게임

## 3. 게임 루프

```
┌─────────────────┐
│    메인 화면      │  ← 포획한 고양이들을 감상 & 꾸미기
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  미니게임 플레이   │  ← 2010년대 패러디 게임
│  (골드 획득)      │
└────────┬────────┘
         │ 골드로 구매
         ▼
┌─────────────────┐
│  여정 꾸러미 구매  │  ← 포획 여정에 필요한 준비물
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  포획 여정 출발   │  ← 로그라이크 (10분+ / 세션)
│  스테이지 클리어  │
│  보스 고양이 등장  │
└────────┬────────┘
    성공 │    │ 실패
         ▼    ▼
   고양이 획득  보상 없음
         │
         ▼
┌─────────────────┐
│  내 고양이 슬롯   │  ← 슬롯 관리 & 확장
└────────┬────────┘
         │
         ▼
      메인 화면으로 복귀
```

## 4. 핵심 시스템

### 4.1 메인 화면 (홈)

- 포획한 고양이들이 메인 화면에서 자유롭게 돌아다님
- **상호작용**:
  - 터치하면 고양이가 반응 (울음소리, 애니메이션 등)
  - 고양이를 끌어다 캣타워, 스크래처 등에 올려두면 상호작용 (추후 적용)
- **꾸미기 기능**:
  - 고양이 의상 꾸미기: 모자, 옷, 액세서리 등
  - 메인 화면 꾸미기: 배경, 가구, 소품 등 (캣타워, 스크래처 포함)
- 메인 화면에서 미니게임, 포획 여정 등 각 콘텐츠로 이동

### 4.2 고양이 포획 (로그라이크)

> 상세 기획: [capture-roguelike.md](capture-roguelike.md)

- **레퍼런스**: 20 Minutes Till Dawn
- **장르**: 탑다운 2D 서바이벌 슈터
- **조작**: 이동 + 회피 + 공격 (고정 맵)
- **플레이 타임**: 5분 / 10분 / 20분 중 선택, 단계가 높을수록 희귀 고양이 출몰 비율 증가
- **무기**: 새총, 탱탱볼, 나무가지, 물총 등 죄책감 없는 장난스러운 무기
- **성장**: 런 내 레벨업 → 스킬 선택 (로그라이크)
- **클리어**: 제한 시간 생존 + 보스 고양이 격파 → 포획 성공

### 4.3 고양이 슬롯 시스템

- 보유 가능한 고양이 수에 **슬롯 제한**이 있음
- 슬롯 확장 방법:
  - **유료 재화** (인앱결제)
  - **희귀 재화** (게임 내 획득)
- 슬롯 제한은 수집 전략과 과금 유도의 핵심 장치

### 4.4 미니게임

2010년대 유행했던 게임들을 고양이 테마로 패러디한 미니게임 컬렉션.
**첫 출시 포함**: 고양이 바운스(바운스볼), 앵그리냥(앵그리버드)
나머지는 업데이트를 통해 점진적으로 추가한다.

- 미니게임 클리어 시 **골드(게임 내 재화)** 를 획득
- 획득한 골드로 **여정 꾸러미**를 구매하여 고양이 포획 여정을 떠날 수 있음
- 미니게임이 포획 여정의 **진입 조건**이자 핵심 재화 수급원

#### 패러디 대상 게임 기반

| 미니게임 후보 | 패러디 대상 | 설명 |
|-------------|-----------|------|
| 고양이 바운스 | 바운스볼 | 공 튀기기 계열 캐주얼 게임 |
| 젤리냥 | 젤리킹 | 젤리 매칭/퍼즐 게임 |
| 고양이 요리사 | 오버쿡드 | 고양이들이 협동하는 요리 게임 |
| 냥의 외출준비 | 슈의 외출준비 | 고양이를 꾸며서 외출 준비시키는 게임 |
| 앵그리냥 | 앵그리버드 | 고양이를 투사체로 날리는 물리 퍼즐 |
| 고양이런 | 쿠키런 | 고양이가 달리는 러닝 액션 |
| 길건너 냥이들 | 길건너 친구들 | 고양이로 길을 건너는 캐주얼 게임 |

## 6. 재화 & 경제 시스템

### 재화 종류

| 재화 | 획득 경로 | 주요 용도 |
|------|----------|----------|
| **골드** (무료 재화) | 미니게임 클리어 보상 | 여정 꾸러미 구매, 일반 꾸미기 아이템 |
| **유료 재화** (보석 등) | 인앱결제 | 슬롯 확장, 프리미엄 꾸미기 아이템 |
| **희귀 재화** | 특수 업적, 이벤트 등 | 슬롯 확장, 한정 아이템 |

### 경제 루프

```
미니게임 플레이 → 골드 획득 → 여정 꾸러미 구매 → 포획 여정 → 고양이 획득
                                                                  ↓
                                              메인 화면에서 감상 & 꾸미기
```

### 인앱결제 아이템 (후보)

| 카테고리 | 아이템 예시 | 설명 |
|---------|-----------|------|
| 슬롯 확장 | 고양이 슬롯 +1 | 보유 가능 고양이 수 증가 |
| 유료 재화 | 보석/캐시 등 | 다목적 프리미엄 재화 |
| 고양이 꾸미기 | 의상, 모자, 액세서리 | 고양이 외형 커스터마이징 |
| 메인 화면 꾸미기 | 배경, 가구, 소품 | 홈 화면 인테리어 |
| 여정 꾸러미 | 프리미엄 꾸러미 | 골드 없이 바로 포획 여정 출발 |

> 구체적인 가격 정책 및 재화 밸런스는 개발 진행 중 설계
